КАК СНЯТЬ МЕРКИ

для короткого женского костюма

Костюм для серфинга, кайтсерфинга можно заказать по размеру ( S, М, 42, 44, 46 ...)
А можно по вашим меркам. На схемах подробно написано какие мерки и как снимать)

КАК СНЯТЬ МЕРКИ

    МЕРКИ:
  1. Общий рост
  2. Обхват груди
  3. Обхват талии
  4. Обхват бедер -

    измеряют
    через наиболее выступающие назад точки ягодиц.

  5. Общая дуга -

    измеряют от точки основания шеи, вниз по прямой, через паховую область, далее вверх по спине, возвращаясь к исходной точке.

  6. Длина плечевого ската -

    измеряют от точки основания шеи по середине плечевого ската, при проведении измерения рука должна быть чуть согнута в локте.

  7. Длина ноги -

    измеряют от паховой области, по внутреннему шву (только для длинного костюма)
